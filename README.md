# tradebot
A bot for automatically trading stocks on Robinhood

Quickstart
====
* Create an `.env` file with the values:
  * `ROBINHOOD_USERNAME`
  * `ROBINHOOD_PASSWORD`
  * `ROBINHOOD_ACCOUNT`
* `npm install`
* `npm start`

Strategies
====
* Different trading strategies are defined in the `strategies` directory and can be switched in `index.js`
